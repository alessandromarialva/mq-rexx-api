Here is the directry structure:
MA95
\bin - contains only z/OS & win-32 executables
\doc - currently empty, latest doc is sent in my previous note
\MVS - z/OS specific parts
\ASM - Assembler stub to implement REXX function pack
\Include - REXX header
\JCL - jobs to build MA95 and to run tests
\samples - REXX samples/tests, self-documented
\src - common sources for all platforms
\Windows - Win/Unix specific code
\Compile - compile command files for Win-32
\Runtests - command files to run tests
\Util - build RXMQCONS.h utility

Note. These deliverables are compiled and tested on
z/OS 1.13 + WMQ 7.0.1
Windows XP + WMQ 7.5 + ooREXX 4.1.1 + gcc 4.4.7
